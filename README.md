# CARE Timeline

## Steps to run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this folder to GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```

2. Go to [Vercel](https://vercel.com) and import the repo.

3. Deploy directly.
